# Install

## Package Management

### npm
``` sh
npm install tailwindcss-vue
```

### yarn
``` sh
yarn add tailwindcss-vue
```

## Usage

### main.js

``` js
import Vue from 'vue';
import TailwindcssVue from 'tailwindcss-vue';

Vue.use(TailwindcssVue);
```
